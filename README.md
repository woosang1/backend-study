# 📚 Backend Study

## 📋 프로젝트 개요
이 프로젝트는 **Kotlin**과 **IntelliJ IDEA**를 활용하여 백엔드 개발을 학습하기 위해 만들어졌습니다.  
**Spring Boot**를 기반으로 다양한 백엔드 기술과 개념을 실습하며, 실제 프로젝트에 적용 가능한 역량을 키우는 것을 목표로 합니다.  

---

## 🛠️ 개발 환경
- **언어**: Kotlin `1.9.22`  
- **IDE**: IntelliJ IDEA Ultimate  
- **프레임워크**: Spring Boot `3.2.4`  
- **빌드 도구**: Gradle
